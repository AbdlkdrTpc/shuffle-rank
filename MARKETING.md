## LinkedIn

### Post 1 — Proje Tanitimi

Icerik ureticileri icin bir arac gelistirdim: Shuffle & Rank.

Sorun basit: Siralama/ranking formati sosyal medyada cok ilgi cekiyor ama bu videolari hazirlamak icin ya video duzenleme yazilimi ya da karmasik araclar gerekiyor.

Cozum: Tarayicida calisan, tek dosyalik bir uygulama.

Nasil calisiyor:
1. Ogelerinizi ekleyin (gorsel veya metin)
2. Basligi ve temanizi secin
3. Karistir butonuna basin
4. Her karti siralama slotuna yerlestirin
5. Kayit modunda videonuzu olusturun

Teknik detaylar:
- Saf HTML + CSS + JavaScript (framework yok)
- Canvas API ile 1080x1920 video render
- MediaRecorder API ile tarayici icinde kayit
- Web Audio API ile ses efektleri
- 116 KB tek dosya

Ucretsiz ve acik kaynak. Herhangi bir kurulum gerektirmiyor.

Canli demo: https://abdlkdrtpc.github.io/shuffle-rank/
Kaynak kod: https://github.com/AbdlkdrTpc/shuffle-rank
